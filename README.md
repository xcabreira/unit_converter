# Conversor de Medidas

Um projeto simples desenvolvido com **HTML**, **CSS** e **JavaScript** para converter diferentes unidades de medida de forma rápida e prática.

## Funcionalidades

* Conversão entre diferentes unidades de medida.
* Interface simples e intuitiva.
* Conversão em tempo real após informar o valor e selecionar as unidades.

## Tecnologias

* HTML5
* CSS3
* JavaScript

## Como usar

1. Abra o arquivo `index.html` ou acesse o link `https://xcabreira.github.io/unit_converter/` em seu navegador.
2. Informe o valor que deseja converter.
3. Selecione a unidade de origem.
4. Escolha a unidade de destino.
5. O resultado da conversão será exibido automaticamente.
